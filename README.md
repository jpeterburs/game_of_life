## Overview
The **Game of Life** is based of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life),
which is a zero-player game.

## Installation
To compile run
```sh
make
```
or use the build in compiler from CodeBlocks
